# Assignment
## 5. Complete HTML5 page
How would the following markup look in HTML5?
+ `<div id="header">`  
  + `<h1>My Page Title</h1>`  
+ `</div>`  
+ `<div id="content">`  
  + `<p>The main content</p>`  
+ `</div>`  
+ `<div id="sidebar">`  
  + `<p>Some stuff in a sidebar</p>`  
+ `</div>`  
+ `<div id="footer">`  
  + `<p>Copyright me</p>`  
+ `</div>`  
Create a complete and valid HTML5 page with required elements.  
Add real content and other HTML elements that you have learned in this unit.

# Solution
La pagina si presenterà così:  
1. Titolo h1 con scritto "My Page Title"
2. Paragrafo con scritto "The main content"
3. Paragrafo con scritto "Some stuff in a sidebar"
4. Paragrafo con scritto "Copyright me"

Ciascun elemento andrà a capo, visto che sono tutti elementi block.

**Nota importante:**  
In questo esempio i valori "header"/"content"/"sidebar"/"footer" sono solo id degli elementi div.  
Questo è utile per il funzionamento della pagina (utilizzo fragment etc.), ma sarebbe best practice in HTML5 utilizzare elementi appositi (header, main, aside, footer), per conferire significato semantico alle sezioni.
